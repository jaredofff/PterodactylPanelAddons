# Ultimate Suite - API Routes Documentation

This directory contains the route definitions for the Ultimate Suite extension.

## 🛣️ API Endpoints

All routes are prefixed with `/api/client` and require authentication.

### Version Manager
- **POST** `/servers/{server}/ultimate-suite/version`
  - Updates the server variables and triggers a reinstall.
  - Body: `{ "type": "paper", "version": "1.20.4" }`

### Player Manager
- **GET** `/servers/{server}/ultimate-suite/players`
  - Returns a list of online players.
- **POST** `/servers/{server}/ultimate-suite/players/command`
  - Executes a command on a specific player.
  - Body: `{ "command": "kick", "player": "Notch" }`

### User Settings
- **POST** `/account/language`
  - Updates the authenticated user's language preference.
  - Body: `{ "language": "es" }`
